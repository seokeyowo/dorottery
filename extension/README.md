# 도로롱 추첨기 헬퍼 확장 프로그램

DC 게시글 페이지에 "🎰 도로롱 추첨" 버튼을 추가해, 클릭 한 번으로 댓글 + 작성자 활동량을 수집해 도로롱 추첨기로 전송합니다.

## 설치 방법 (Chrome / Edge / Whale)

1. 이 `extension` 폴더를 원하는 위치에 복사 (예: `C:\dororong-extension\`)
2. 브라우저 주소창에 다음 입력:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Whale: `whale://extensions`
3. 우측 상단 **개발자 모드** 켜기
4. **압축해제된 확장 프로그램을 로드** 클릭
5. 위에서 복사한 `extension` 폴더 선택
6. 목록에 "도로롱 추첨기 헬퍼"가 활성화되면 끝

## 사용 방법

1. 추첨할 DC 게시글 페이지 접속
2. 우측 하단에 **🎰 도로롱 추첨** 버튼이 뜸
3. 클릭 → 자동 수집 → 새 탭에서 추첨기 열림
4. 추첨기에서 필터 설정 후 추첨 시작

## 작동 원리

- Chrome 확장의 content script가 DC 페이지에 실행됨
- DC 댓글 API와 갤로그 페이지를 **내 브라우저 IP**로 호출 (차단 회피)
- 수집된 데이터는 URL hash로 추첨기 사이트에 전달
